解析规则说明
解析规则:
  • 优先级1: 如果同时有[]和<>，直接返回<>中的内容
  • 优先级2: 如果只有<>，返回<>中的内容
  • 优先级3: 如果只有[]，返回最后一个[]中的内容
  • 优先级4: 如果都没有，返回原始消息
示例:
项目创建失败: [命令传输对象校验失败]<项目日志框架不能为空> → 项目日志框架不能为空
项目创建失败: [命令传输对象校验失败]<请检查输入参数> → 请检查输入参数
